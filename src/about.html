About PixelPaws

What is PixelPaws?

PixelPaws is a virtual pet companion designed to bring joy and comfort to students and anyone who needs a digital friend. Your PixelPaws pet reacts to real-world weather conditions, creating a dynamic and engaging experience.

Whether you're taking a study break or just need some companionship, your PixelPaws friend is always there to brighten your day with its charming personality and weather-appropriate moods.

Features

  • Weather-Reactive Pet: Your pet's mood and environment change based on real-world weather conditions
  • Full Customization: Personalize your pet's appearance including ears, eyes, face, colors, and tail
  • Interactive Activities: Play, feed, pet, and care for your virtual companion
  • Responsive Design: Enjoy PixelPaws on any device - desktop, tablet, or mobile
  • Persistent Data: Your pet's name and customizations are saved for future visits

Technology

PixelPaws is built using modern web technologies:

  • HTML5: Semantic markup for accessibility and SEO
  • CSS3: Responsive design with custom animations and transitions
  • JavaScript: Interactive functionality and API integration (coming in future updates)
  • Weather API: Real-time weather data integration (planned feature)

About the Developer

PixelPaws was created by Bridget Kimball as a final project for CS 343 at James Madison University.

The goal was to create an engaging, accessible web application that combines creativity with technical skills, while providing a useful tool for stress relief and companionship for students.

Project Goals

This project demonstrates proficiency in:

  • Semantic HTML and web accessibility standards
  • Responsive CSS design with media queries
  • User interface and user experience design
  • Version control with Git and GitHub
  • Web deployment and hosting
  • API integration and data handling
